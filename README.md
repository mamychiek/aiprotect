Automated APK/IPA scanner for insecure data storage, HTTPS misconfigurations, and weak encryption. Generates developer‑ready reports.

2. Small Business Security Monitoring Agent
Real‑time log analysis for small enterprises. Detects brute‑force, port scans, and anomalies. Sends instant alerts via WhatsApp & Telegram with remediation steps.

3. Cloud Misconfiguration Scanner Agent
Scans AWS/Azure/GCP (or Terraform) for public buckets, over‑permissive IAM, and open security groups. Opens tickets with fix suggestions.

4. Phishing Email Analysis Agent
Analyzes EML files/headers using SPF/DKIM/DMARC, URL blocklists, and attachment hashes. Returns verdict + IOCs + YARA rules.

5. Insider Threat Detection (UEBA) Agent
Builds user behavior baselines from AD/VPN/file logs. Flags impossible travel, bulk downloads, and privilege escalation. Integrates with Slack & LDAP.
